Keeping track of movies with Emacs and Org mode

January 31, 2021

Introduction

A while back, I decided to start cataloging the movies I’ve seen. This started as a simple table in Emacs Org mode. Below is an example of the table’s structure:

| Movie Title | Release year | Genre                        | Rating     | Notes            |
|-------------+--------------+------------------------------+------------+------------------|
| Up          |         2009 | Animation, Adventure, Comedy | 8.5 / 10.0 | Lots of balloons |
| WALL-E      |         2008 | Animation, Adventure, Family | 8.5 / 10.0 | Don't litter!    |
| Toy Story   |         1995 | Animation, Adventure, Comedy | 9.0 / 10.0 |                  |

The thing that annoyed me the most about the above setup was having to manually enter the metadata for each movie. I wanted to simply add the title and/or release year and have the rest of the information filled in for me. I googled around for a solution and found this Gist which demonstrates how to import movie metadata from OMDB (“The Open Movie Database”) into Org mode (specifically, a nicely formatted IMDB link with some metadata attached to it). Although not exactly what I want to do, it did introduce me to what OMDB was and how one can query it with a little bit of Elisp code.

Using OMDB’s API

Using OMDB’s API is pretty simple. First, sign-up here to receive an API key. The free-tier offers 1000 requests per day, which should be plenty for this project.

Once you have a key, you can start making requests. For example, running:

GET http://omdbapi.com/?t=Up&apiKey=[YOUR KEY]

Yields:

{
   "Title":"Up",
   "Year":"2009",
   "Rated":"PG",
   "Released":"29 May 2009",
   "Runtime":"96 min",
   "Genre":"Animation, Adventure, Comedy, Family",
   "Director":"Pete Docter, Bob Peterson(co-director)",
   "Writer":"Pete Docter (story by), Bob Peterson (story by), Tom McCarthy (story by), Bob Peterson (screenplay by), Pete Docter (screenplay by)",
   <snip>
}

You can also optionally specify the year with the y query parameter. This will help disambiguate movies with the same title that were released in different years. If two movies have the same name and were both released in the same year1 - well, lets now worry about that for now 😄.
